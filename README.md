## PyInvoke for IntelliJ IDEs
<!-- Plugin description -->
Provides [PyInvoke](https://github.com/pyinvoke/invoke) support to all IntelliJ-based IDEs.

Execute and debug your PyInvoke tasks directly within your IDE.
